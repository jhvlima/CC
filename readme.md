# Compiladores

## 🚀 Como Usar Agora (Simplificado)

**Para qualquer pessoa fazer git clone:**

```bash
git clone <url-do-repo>
cd CC
./setup.sh
cd lab1
```

### 🔧 O que o `setup.sh` Faz Automaticamente

- ✅ Verifica Java/JDK instalado
- ✅ Verifica Make disponível  
- ✅ Baixa ANTLR se necessário
- ✅ Testa build completo
- ✅ Valida execução do lexer
- ✅ Mostra instruções finais
